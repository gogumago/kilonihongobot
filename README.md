# 🗾 일본어 한 입 — 텔레그램 학습 봇

한국인이 이미 아는 외래어(サウナ·カメラ·コーヒー...)와 기초 단어로
**히라가나·가타카나를 자연스럽게 익히는** 텔레그램 봇입니다.

## 구성
- **단어 300개**: 가타카나 외래어 173개 + 히라가나 기초 단어 127개
  - 모든 단어에 한글 발음 + **글자별 발음 분해** 자동 표시
- **문장 300개**: 배운 단어가 그대로 들어간 초단문, **핵심 문법 37종** 순환
  - です/ですか/じゃないです/でした, あります·います, ください,
    すきです/ほしいです, ます동사, 조사 は·が·を·に·で·の·も,
    い형용사·な형용사, ましょう, ています 등
- **발송 스케줄**: 매일 08·10·12·14·16·18·20시 (2시간 간격, 메시지당 2레슨)
- **총정리**: 매일 밤 21:30, 그날 배운 단어·문장·문법 요약
- 300개를 다 돌면 자동으로 처음부터 복습 (N회차 표시)

## 설치 & 실행
1. 텔레그램에서 **@BotFather** 검색 → `/newbot` → 봇 이름 설정 → **토큰 복사**
2. Python 3.10+ 환경에서:
   ```bash
   pip install "python-telegram-bot[job-queue]"
   ```
3. 토큰 설정 후 실행:
   ```bash
   export BOT_TOKEN="여기에_토큰"     # 윈도우: set BOT_TOKEN=토큰
   python bot.py
   ```
4. 텔레그램에서 내 봇을 찾아 `/start` 전송 → 끝!

> ⚠️ 봇이 메시지를 계속 보내려면 **프로그램이 24시간 켜져 있어야** 합니다.
> 집 PC를 켜두거나, 무료/저가 클라우드(Oracle Cloud Free, 라즈베리파이,
> PythonAnywhere, Railway 등)에 올려두는 걸 추천합니다.

## 명령어
| 명령어 | 기능 |
|---|---|
| `/start` | 구독 시작(재개) + 첫 레슨 |
| `/now` | 지금 바로 레슨 1개 받기 |
| `/today` | 오늘 배운 내용 총정리 다시 보기 |
| `/progress` | 진도 바 확인 |
| `/stop` | 발송 일시 중지 |

## 커스터마이징
`bot.py` 상단 설정만 바꾸면 됩니다:
```python
LESSON_HOURS = [8, 10, 12, 14, 16, 18, 20]  # 발송 시각
SUMMARY_TIME = dt.time(21, 30, tzinfo=TZ)   # 총정리 시각
ITEMS_PER_MESSAGE = 2                        # 메시지당 레슨 수
```
단어를 추가/수정하려면 `data/words.py` 를 고친 뒤:
```bash
python build_lessons.py   # 문장·발음 자동 재생성
```
